# EquiThera Hub

EquiThera Hub is a healthcare support application built using Flutter.

## Features

- Role-based login system
- Common Dashboard
- Transgender Healthcare Dashboard
- Scan Prescription
- Drug Interaction Checker
- Mental Health Bot
- AI Medical Helpline
- Emergency SOS
- Hormone Therapy Tracker
- Health Records
- Risk Alerts

## Tech Stack

- Flutter
- Dart
- Android Studio
